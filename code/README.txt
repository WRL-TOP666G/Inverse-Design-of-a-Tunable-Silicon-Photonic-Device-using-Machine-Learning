=======    
Reference: SPINS-B (https://github.com/stanfordnqp/spins-b)
=======

Installation
-------------
* Download file 14W
* Go into folder 14W
* Use following comment line to install spins-b 

pip3 install -e .

Start training 
-------------
* In 14W/main
* Run training.py file to start training 

python3 training.py

Visualization
-------------
* In 14W/main
* Use Visualize.ipynb to plot the training results
